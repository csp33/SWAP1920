En esta tarea veremos qué es LFS, sus ventajas y la arquitectura que plantea.

Lustre File System (LFS) es un sistema de archivos distribuidp paralelo cuyo objetivo principal es servir en entornos con hardware de altas prestaciones (HPC, High Performance Computing). Es el sistema de archivos más utilizado en el top 500 HPCs. Su ventaja principal es la escalabilidad que proporciona, permitiendo transmisiones de archivos muy grandes entre muchos nodos.\\

La arquitectura de LFS se basa en las siguientes capas:
\begin{itemize}
	\item OSS (Object Storage Servers). Gestionan el acceso E/S y las peticiones de red.
	\item OST (Object Storage Targets). Dispositivos de almacenamiento que contienen los datos en sí.
	\item MDS (MetaData Server). Mapea la ubicación de cada archivo para poder dirigir las peticiones.
	\item MDT (MetaData Target). Almacena los metadatos de cada archivo (nombre, directorio, permisos, etc.)
\end{itemize}
